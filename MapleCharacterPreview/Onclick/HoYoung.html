천방지축 도사, 호영

호영은 부적 도력과 두루마리 도력이라는 이중 게이지 시스템을 사용하는 게이지 직업입니다.


하지만 선기 : 영약 태을선단 덕분에, 다른 게이지 직업에 비해 쉬운 게이지 운용 난이도를 보여줍니다.


또한 호영은 강력한 극딜, 좋은 사냥 능력, 뛰어난 기동성과 최상급의 생존 유틸, 쉬운 딜사이클을 가진 팔방미인 직업입니다.


하지만 보스전에서 유지해야 하는 스킬이 무려 4가지나 된다는 점과 높은 조작 난이도, 높은 코어 강화 난이도, 주력기의 선/후딜 문제 등등의 단점이 존재합니다.

1차 전직 - 호영

스킬 설명

부적 도력 - 패시브

호영의 첫 번째 게이지. 천/지/인 속성 도술 적중 시 부적 도력이

충전되고, 부적 도력을 사용하여 부적 도술을 사용할 수 있다.

여의선 : 인 - 액티브

[인 속성 도술] 호영의 사냥 주력기.

거대하게 변화시킨 부채를 휘두른다.

마봉 호로부 - 액티브

[부적 도술]

호리병으로 적을 흡수했다가 발사한다.

초상비 - 액티브

호영의 트리플 점프. 가벼운 몸놀림으로 도약한다.

근두은 - 액티브(키다운)

호영의 공중 부양형 이동/유틸기. 일정 시간 비행을 할 수 있다.

2차 전직 - 호영

스킬 설명

토파류 : 지 - 액티브

[지 속성 도술] 호영의 보스 주력기. 땅을 차서

지면을 타고 달리는 기운을 날린다.

토파류 : 허/실 - 액티브

토파류 : 지의 연계 스킬. 시전시 피해를 입힌다.

토파류 : 실 : 토파류 : 지 시전 중 방향키와 함께 재사용시 돌진한다.

토파류 : 허 : 토파류 : 실로 연계하지 않을 시 자동적으로 사용된다.

환영 분신부 - 액티브(버프)

[부적 도술]

일정 시간 동안 적을 공격 시 일정 확률로

분신이 나와 적을 공격한다.

3차 전직 - 호영

스킬 설명

두루마리 도력 - 패시브

호영의 두번 째 게이지. 천/지/인 도술 사용 및 부적 도술 사용 시

두루마리 도력이 충전되고, 두루마리 도력을 사용하여

두루마리 도술을 사용할 수 있다.

파초붕 : 천 - 액티브

[천 속성 도술] 호영의 유틸기. 공중으로

도약한 뒤 바람을 일으켜 공격한다.

파초풍 : 허/실 - 액티브

파초풍 : 천의 연계 스킬. 시전시 적을 밟아 피해를 입힌다.

파초풍 : 실 : 파초풍 : 천 시전 중 방향키와 함께 재사용시

다시 한번 공중으로 도약한다.

파초풍 : 허 : 파초풍 : 실로 연계하지 않을 시 자동적으로 사용된다.

지진쇄 : 지 - 액티브

[지 속성 도술] 지면을 강하게 내리찍어 주변에 피해를 입힌다.

지진쇄 : 허/실 - 액티브

지진쇄 : 지의 연계 스킬. 시전시 피해를 입힌다.

지진쇄 : 실 : 토파류 : 지 시전 중 방향키와 조합하여 순간이동한다.

지진쇄 : 허 : 지진쇄 : 실로 연계하지 않을 시 자동적으로 사용된다.

추적 귀화부 - 액티브(버프)

[부적 도술]

도깨비불을 소환하여 적을 추적하여 공격한다.

권술 : 미생강변 - 액티브

[두루마리 도술]

일반 몬스터에게 시전 시 미물로 변이 및 방어율을

감소시키고, 보스 몬스터에게는 방어율만 감소시킨다.
